##  Como Rodar:

1.  **Clone e Instale:**
    ```bash
    git clone [https://github.com/Samuel-Jordano/pethub-2.git]
    pip install -r requirements.txt
    ```

2.  **(Banco de Dados):**
    ```bash
    python manage.py migrate
    python manage.py createsuperuser
    ```

3.  **Decole:**
    ```bash
    python manage.py runserver
    ```

Acesse **`http://127.0.0.1:8000/`**
